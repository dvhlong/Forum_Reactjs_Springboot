# Đoàn Văn Hoàng Long - Tech Forum

## Introduction

This project is a forum system built with two main components:
- Backend: Spring Boot
- Frontend: ReactJS

## Purpose

- This is a university graduation project, implemented to improve skills in developing modern web applications using ReactJS & Spring Boot.
- The project provides practice in building APIs, security, user interface, and fullstack application deployment.

## Features

- User registration & login
- Topic creation, posting, commenting
- Search and categorize topics
- Personal account management
- Other forum management functionalities

## Deployment & Usage

Netlify (ReactJS)

Heroku (Spring Boot, PostgreSQL)

Link open app: https://phenomenal-madeleine-18dbfe.netlify.app (Shortened link: bit.ly/sorais599_Forum)

Currently, the project is not deployed online because Heroku has moved to a paid model. You can run the application locally as follows:

### Backend (Spring Boot)

1. Install Java 11+ and Maven.
2. Clone this repository and navigate to the backend folder.
3. Adjust your database configuration in `application.properties`.
4. Run:
   ```
   mvn spring-boot:run
   ```

### Frontend (ReactJS)

1. Go to the frontend folder of the project.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the application:
   ```
   npm start
   ```
4. The app will run at `http://localhost:3000`.
